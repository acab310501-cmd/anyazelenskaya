Сайт персонального фитнес-тренера (Аня Зеленская)

Задача:
Создать сайт, который с первого взгляда вызывает доверие и желание записаться. Клиент — молодой сертифицированный тренер с 10-летним опытом, который хочет работать с премиальной аудиторией.

Решение:
Разработала минималистичный лендинг с акцентом на личность тренера. Главный экран — крупное имя, фото, кнопка и 3 ключевых преимущества. В разделе «Обо мне» — фотогалерея и текст, показывающий экспертность. Блок цен выполнен в виде «стеклянных» карточек с детальным описанием каждого пакета.

Реализована форма записи: после заполнения данных клиент видит модальное окно с конфетти, а тренер получает уведомление в Telegram с полной информацией о заявке.

Особенности:

Адаптив под все устройства

Тёмная тема по клику

Автопереключающиеся советы (50 штук)

Аккордеон FAQ

Слайдер отзывов

Плавные анимации по скроллу

Технологии:
HTML, CSS (Flexbox, Grid), JavaScript, Telegram Bot API, GitHub Pages

Результат:
Сайт принимает заявки онлайн, освобождая тренера от переписки. За первый месяц работы пришло 10 заявок. Клиенты отмечают «дорогой» внешний вид и удобство записи.
